Thought for 2s# 拼豆像素画生成器 - 产品需求文档

## 1. 产品概述

**产品名称**: 拼豆像素画生成器

**产品定位**: 一个帮助用户将普通照片转换为拼豆像素画的在线工具，支持自定义编辑、作品展示和社区分享。

**核心价值**:

- 简化拼豆艺术创作流程
- 提供直观的像素化编辑工具
- 构建创意社区展示平台


## 2. 目标用户

**主要用户群体**:

- 拼豆手工艺爱好者
- 像素艺术创作者
- DIY手工艺品制作者
- 艺术教育工作者


**用户需求**:

- 快速将照片转换为可制作的拼豆图案
- 精确控制像素化效果
- 分享和发现创意作品


## 3. 核心功能模块

### 3.1 首页模块

**功能描述**:

- 展示产品核心价值主张
- 提供快速上传入口（模态窗口）
- 优雅的视觉设计，带渐变背景和装饰图案
- 滚动指示器引导用户探索更多内容


**交互流程**:

1. 用户进入首页看到欢迎界面
2. 点击"开始创作"触发上传模态窗口
3. 点击"发现"按钮平滑滚动到创意豆坊区域
4. 查看滚动提示动画


### 3.2 图片上传与转换模块

**功能描述**:

- 模态窗口式上传界面
- 支持拖拽和点击上传
- 自动转换为像素画预览
- 跳转到编辑器页面


**技术要求**:

- 支持 JPG、PNG 格式
- 图片大小限制：10MB
- 即时预览转换效果


### 3.3 像素画编辑器模块

**功能描述**:

- 可调节像素化程度（滑块控制）
- 颜色数量控制
- 实时预览效果
- 导出功能


**控制参数**:

- 像素大小：1-50
- 颜色数量：8-256
- 实时更新画布


### 3.4 创意豆坊（发现）模块

**功能描述**:

- 展示精选示例作品
- 网格布局展示
- 卡片悬停动画效果:悬停后显示像素图片
- 点击可查看详情或复制创作


**展示内容**:

- 示例像素画作品
- 作品标题
- 交互动画


### 3.5 作品集模块

**功能描述**:

- 用户上传完成的拼豆实物照片
- 添加作品介绍
- 网格展示所有作品
- 支持返回首页


**上传界面**:

- 图片预览功能
- 作品标题输入
- 作品描述输入（多行文本）
- 提交按钮


**展示效果**:

- 标题动画效果
- 返回按钮
- 响应式网格布局
- 卡片悬停效果


## 4. 用户流程

### 主流程：

```plaintext
首页 → 点击"开始创作" → 上传模态窗口 → 上传图片 → 编辑器 → 调整参数 → 导出作品
```

### 发现流程：

```plaintext
首页 → 点击"发现" → 滚动到创意豆坊 → 浏览示例 → 点击示例 → 查看详情
```

### 作品集流程：

```plaintext
首页 → 点击"作品集" → 作品集页面 → 点击上传 → 填写信息 → 提交 → 展示在作品集中
```

## 5. 设计规范

### 5.1 色彩系统

- 主色调：紫色渐变 (`#8B5CF6` → `#EC4899`)
- 背景色：深色主题 (`#0a0a0a`, `#171717`)
- 边框色：柔和灰色 (#333)
- 文字色：白色/浅灰色


### 5.2 字体规范

- 标题：大字号，粗体
- 正文：中等大小，常规字重
- 按钮：中等字号，适中字重


### 5.3 动画效果

- 页面加载淡入动画
- 按钮悬停缩放效果
- 卡片悬停提升效果
- 滚动指示器持续动画
- 标题文字动画


### 5.4 响应式设计

- 移动端：单列布局
- 平板：双列布局
- 桌面：三列或四列布局


## 6. 技术架构

### 6.1 技术栈

- 框架：Next.js 16
- 样式：Tailwind CSS v4
- UI 组件：shadcn/ui
- 图片处理：Canvas API


### 6.2 页面结构

```plaintext
/                    # 首页（包含创意豆坊区域）
/editor/[id]         # 编辑器页面
/gallery             # 作品集页面
```

### 6.3 组件架构

- 上传模态组件
- 图片编辑器组件
- 作品卡片组件
- 滚动指示器组件

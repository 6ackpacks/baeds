## 🎯 关键实现细节

### 吉祥物动画
```typescript
// 眨眼动画
<motion.g animate={isHovered ? { y: [0, 2, 0] } : { y: 0 }}>
  {/* 眼睛 */}
</motion.g>

// 手臂挥动
<motion.g
  animate={isHovered ? { rotate: [-5, 10, -5] } : { rotate: 0 }}
  transition={{ duration: 0.6, repeat: Infinity }}
  style={{ transformOrigin: "170px 120px" }}
>
  {/* 手臂 */}
</motion.g>

// 金色粒子
{isHovered && (
  <motion.circle
    animate={{ y: [-10, 10, -10], opacity: [0, 1, 0] }}
    transition={{ duration: 1.5, repeat: Infinity }}
  />
)}
```

// 上传图片
